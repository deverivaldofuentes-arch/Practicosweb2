¿Cómo Usarlo?

Integra este sistema en tu proyecto en pocos minutos

1

Incluye los Archivos

Añade los archivos CSS y JS a tu proyecto

<link rel="stylesheet" href="modal-system.css">
2

Añade el HTML

Estructura básica del modal en tu HTML

<div class="modal" id="miModal">...</div>
3

Activa el Modal

Añade el atributo data-modal a cualquier botón

<button data-modal="miModal">Abrir</button>
